# Repository Scope

## What This Repository Is

This repository exists to hold the planning and design documentation for **Astroship**.

Its current purpose is to document:

- the game vision
- player fantasy
- core progression systems
- social and fleet systems
- monetization direction
- MVP scope
- live-service planning

## What This Repository Is Not

This repository is **not** currently intended to be:

- an open-source game project
- a public contribution project
- a finalized production codebase
- a community template

## Documentation-First Intent

At this stage, the goal is to create and maintain strong documentation before committing to full game implementation.

That means the priority is:

1. clarifying the product
2. aligning on systems
3. documenting scope and risks
4. preparing for eventual production work

## Proprietary Status

**Astroship is proprietary.**

The design materials in this repository are not open source and should be treated as private product documentation unless the project owner explicitly publishes a different license later.
